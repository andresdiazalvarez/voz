# Voz

Aplicación nueva y limpia para controlar extintores.

## Incluye

- Importar Excel y añadir solo registros nuevos.
- Filtrar por edificio, número SYCo y nº serie.
- Ordenar por edificio o número SYCo.
- Editar registros y añadir nuevos.
- Modelo con lista de opciones y escritura libre.
- Fechas con escritura libre y lista de años.
- Señal de 2017 a 2026.
- Defectos múltiples.
- Dos fotos por registro, con opción de eliminar/cambiar.
- Descargar Excel con datos, defectos y fotos.

Los datos y fotos se guardan en el navegador/móvil mediante IndexedDB.
